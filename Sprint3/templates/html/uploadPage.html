Upload CSV files for creating exam schedule for relevant departments of the faculty.

Please ensure that the CSV files you upload adhere to the following format in order:


  • Section: Indicates the section of the course.
  • Course_Code: Specifies the code of the course.
  • Course_Name: Specifies the name of the course.
  • Number_of_Students: Indicates the number of students enrolled in the course.
  • T+U: Indicates the total number of weekly hours for the course.
  • AKTS: Specifies the credit hours of the course according to the European Credit Transfer and Accumulation System (ECTS).
  • Course_Environment: Specifies the environment of the course (e.g., online or face-to-face).
  • Department: Indicates the department to which the course belongs.
  • Class: Specifies the class of the course.
  • Lecturer: Specifies the name of the lecturer teaching the course.

Upload a CSV file containing classroom information and capacities:


Please ensure that the CSV files you upload for classroom information adhere to the following format:


  • Floor_Number: Specifies the floor number where the classroom is located.
  • Classroom_Number: Specifies the classroom number.
  • Capacity: Indicates the capacity of the classroom.
  • Usage: Specifies how many days the classroom is used for program.
